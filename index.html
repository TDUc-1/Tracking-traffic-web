Quick Setup — 2 Steps

Step 1: Create a Firebase project at console.firebase.google.com, enable Firestore Database, and copy your config into tracker.js and the FIREBASE_CONFIG block below in this file.

Step 2: Add <script src="tracker.js"></script> before the </body> tag on every page you want to track. Then open this dashboard — data appears here in real time.
